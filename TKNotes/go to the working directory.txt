go to GitHub, make repo with same name(or whatever name i guess) 
copy ssh key e.g. git@github.com:leet0727/whateverprojectname.git

git branch -M main //makes this the main branch 
git push -u origin main //pushes files in that directory into the main branch 

git checkout -b experiment1 //Create a new branch (experiment universe) and switch to it

git push --set-upstream origin experiment1 //Publish the branch to GitHub (first time only)

git merge experiment1 //merges experiment1 to main 
git push


git branch -d experiment1 //deletes locally 
git push origin --delete experiment1 //deletes on GitHub


git log --graph --oneline --all //shows all branches
